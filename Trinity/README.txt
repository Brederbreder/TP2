O trinity é um estilo de programação aonde, o programa é dividido em três componentes: o modelo, a vista e o controlador. O modelo representa os dados do aplicativo; a visualização representa uma representação específica dos dados; o controlador fornece entrada / saída, preenche / atualiza o modelo e invoca a visualização correta.

Essa parte do trabalho foi realizada em codigo java, as classes estão dentro do pacote trinity. Sendo assim para compilar o programa deve-se primeiro entrar no diretorio aonde estão os texto-exemplos (como o moby-dick.txt) e o 
pacote trinity.
Em seguida se realiza as seguintes ações:

	$ javac trinity/*.java 

Este comando compilara todos arquivos .java dentro do pacote trinity, e para executar o programa:

	$ java trinity.MainClass arquivo.txt"

Aonde o 'arquivo.txt' é um dos texto-exemplos necessario para começar o programa.
